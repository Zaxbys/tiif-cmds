                                                                           Utils Commands
                                                                            

avatar - returns the pinged user's avatar
    Usage: avatar [user]
ping - returns the ping of the Discord's API
    Usage: ping
    Aliases: latency
serverinfo - returns the information of the guild you're in
    Usage: serverinfo
    Aliases: si, guildinfo, ginfo, gi
